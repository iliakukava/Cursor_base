# Презентация TeleFlow (статический сайт)

Папка: только **HTML + CSS + JS**, без сборки и npm. 13 слайдов, оптимизировано под питч 5–7 минут.

## Как открыть

- Дважды кликнуть по `index.html`, **или**
- Локальный сервер (нужен только для Google Fonts без CORS-warning в некоторых браузерах):

```bash
cd presentation
python -m http.server 8765
```

Открыть: `http://127.0.0.1:8765/`

## Управление

- **Прокрутка** колёсиком/тачпадом — секции прилипают по высоте экрана (`scroll-snap`).
- **Стрелки** <kbd>↑</kbd>/<kbd>↓</kbd>/<kbd>←</kbd>/<kbd>→</kbd>, <kbd>Page Up/Down</kbd>, <kbd>Space</kbd> — между слайдами.
- <kbd>Home</kbd>/<kbd>End</kbd> — в начало/конец.
- **Точки слева** (на мобильных — снизу) — переход к разделу.
- Жесты свайпа — переключение слайдов.

### Инструменты спикера

Панель в правом верхнем углу:

- **Счётчик слайдов** — `NN / NN`.
- **Встроенный таймер** — 5:00 зелёный, 7:00 жёлтый, дальше красный.
  - <kbd>T</kbd> — старт/пауза таймера.
  - <kbd>F</kbd> — полноэкранный режим.
  - <kbd>P</kbd> — печать / экспорт в PDF (страницы 1440×900).

## Что править под себя

- Цифры экономии на слайде 6 и разбивка времени на слайде 10 — замени под свои реальные замеры.
- Имя бренда TeleFlow, email и Telegram — в `index.html` (слайды 1 и 13).
- Цвета, шрифты, радиусы — в `:root` в `style.css`.

## Печать в PDF

`Ctrl/Cmd+P` → Save as PDF, страницы 1440×900 настроены через `@media print` в `style.css`. Навигация, таймер и прогресс-бар при печати скрыты.
