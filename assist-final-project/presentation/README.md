# Презентация TeleFlow · Dossier № 01

Статический HTML-deck из 12 слайдов, в editorial-стиле на тёмно-синей палитре.
Никакой сборки, npm и CDN-шаблонов — только `index.html`, `style.css`, `app.js` и три шрифта из Google Fonts.

## Дизайн-концепция

Основная идея — «цифровое досье журнала». Каждый слайд — **разворот журнала**: номер (римская цифра) на рельсе слева, секция в шапке, асимметричная двухколоночная сетка, serif-заголовки (Instrument Serif) + sans-тело (Inter) + моно для цифр и кода (JetBrains Mono). Один яркий акцент — аквамарин (`#7befd9`), один приглушённый (`#e9d6a6`), всё остальное — оттенки тёмно-синего. Тонкая film-grain поверх фона убирает ощущение «пластиковой» AI-генерации.

## Как открыть

```bash
# Двойной клик по index.html, или:
cd presentation
python -m http.server 8765
# → http://127.0.0.1:8765/
```

## Управление

- **Прокрутка** колёсиком/тачпадом — секции прилипают (`scroll-snap`).
- <kbd>↑</kbd>/<kbd>↓</kbd>/<kbd>←</kbd>/<kbd>→</kbd>, <kbd>Page Up/Down</kbd>, <kbd>Space</kbd> — между слайдами.
- <kbd>Home</kbd>/<kbd>End</kbd> — к первому/последнему.
- <kbd>F</kbd> — полноэкранный режим.
- <kbd>P</kbd> — печать / экспорт в PDF (1440×900).
- Свайпы по горизонтали — переключение.
- Боковой вертикальный навигатор справа — переход к конкретной секции.

В правом верхнем углу — счётчик `NN/12`. В левой рельсе — римская цифра текущего слайда.

## Что править под себя

- Цифры на слайде V и разбивка часов на слайде IX — под свои замеры.
- Имя бренда `TeleFlow` — в `index.html` (рельса, обложка, финальная рулёвая линия).
- Акцентные цвета (`--aqua`, `--sand`, `--sky`) — в `:root` в `style.css`.
- Шрифты — в `<link>` HTML и в `:root` CSS (три переменные `--font-*`).

## Печать в PDF

`Ctrl/Cmd+P` → Save as PDF. `@media print` настроен на страницы 1440×900,
рельса / прогресс / навигация / счётчик при печати скрыты.

## Доступность

- Работает с клавиатуры (полная навигация).
- Есть `skip-link`, `role="progressbar"`, `aria-label` на всех ключевых регионах.
- `prefers-reduced-motion` — отключает анимации, film-grain, pulse схемы.
- Контраст текста к фону проверен — все оттенки серо-голубого держат WCAG AA на основном тексте.
