    TeleFlow

    Дайджесты, тематическая база и черновики постов из Telegram — с LLM и под вашим контролем.

    Прокрутка или стрелки ↑ ↓ · Home / End

    Проблема

    Слишком много каналов и папок: непрочитанное копится, важное теряется, на ручную выжимку уходит время каждый день.

    Что это

    Локальный Telegram userbot (Telethon): команды в ЛС, сохранение материалов по темам, генерация текста через OpenRouter.

    • /digest — дайджест из выбранных папок
    • Forward поста → выбор темы → база знаний
    • /write <тема> — пост из накопленного
    • /stats — сколько постов по темам

    Схема потока

    Разметка дайджеста и синтез /write — там, где ИИ даёт выигрыш по времени.

    Польза

    Один контур: обзор непрочитанного, архив по темам, черновик поста под свою ленту или канал — без ручного копипаста по десятку чатов.

    Контроль и безопасность

    • Доступ только для пользователей из whitelist
    • Режим DRY_RUN — превью без записи в канал и state
    • Длинные ответы — отправка частями (лимиты Telegram)
    • Данные локально: data/knowledge/, state.db

    Время на проект
