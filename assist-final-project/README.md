## Про сам проект (AssistHQ)

**Одна строка:** Telegram-бот, который работает как младший ассистент предпринимателя — собирает голосовые и сообщения, ведёт контекст клиентов и сам готовит ответы.

**Ключевые идеи:**

- Весь стек проекта — ровно то, что дал курс: Cursor, `.mdc` правила, агенты, aiogram, OpenAI API, Whisper, Google Sheets, деплой на VPS.
- Каждый модуль курса оставил конкретный артефакт в проекте — это сознательный выбор, а не случайность.
- Цель — не отчитаться, а попасть в команду АССИСТ+ и развивать проект как продукт.

**Метрики, которые звучат в питче:**

| Метрика | Значение |
| --- | --- |
| Экономия времени собственника | ≈ 2 ч/день, до 40 ч/мес |
| Себестоимость 100 сообщений | ≈ 60 ₽ (GPT-4o-mini + Whisper) |
| Время на создание проекта | ≈ 38 часов за 5 недель |
| Ускорение vs ручной путь | ×6 |

## Как показывать презентацию

Presentation — один самодостаточный HTML-файл, без внешних зависимостей и CDN. Работает офлайн.

1. Открой `presentation.html` в любом современном браузере (Chrome / Safari / Firefox).
2. Нажми `F` — полноэкранный режим.
3. Нажми `T` — встроенный таймер спикера (зелёный до 5:00, жёлтый до 7:00, красный дальше).
4. Навигация:
   - `→` / `Space` / `PageDown` — следующий слайд
   - `←` / `PageUp` — предыдущий
   - `Home` / `End` — в начало / в конец
   - `P` — печать / экспорт в PDF (страницы 1440×900)
   - Жесты свайпа на тачскрине — переключение слайдов
