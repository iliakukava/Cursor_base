# Все горячие клавиши Cursor — полный справочник (Windows)

> Только **Windows**. Сочетания как в VS Code / Cursor на ПК.

---

## 🤖 AI-функции (самое важное)

| Действие | Сочетание |
|----------|-----------|
| AI Chat (боковая панель) | `Ctrl+L` |
| Добавить выделенное в Chat | `Ctrl+Shift+L` |
| Composer (мультифайловый) | `Ctrl+I` |
| Inline Edit (быстрая правка) | `Ctrl+K` |
| Принять Inline Edit | `Ctrl+Shift+K` |
| Tab (принять автодополнение) | `Tab` |
| Отклонить автодополнение | `Esc` |
| Следующее предложение | `Alt+]` |
| Новый чат (в панели чата) | `Ctrl+N` |

---

## 📁 Файлы и навигация

| Действие | Сочетание |
|----------|-----------|
| Быстрое открытие файла | `Ctrl+P` |
| Новый файл | `Ctrl+N` |
| Сохранить | `Ctrl+S` |
| Сохранить всё | `Ctrl+Alt+S` |
| Закрыть файл | `Ctrl+W` |
| Переключение вкладок | `Ctrl+1` … `Ctrl+9` |
| Следующая вкладка | `Ctrl+Tab` |
| Показать Explorer | `Ctrl+Shift+E` |

## ✏️ Редактирование

| Действие | Сочетание |
|----------|-----------|

| Повторить | `Ctrl+Shift+Z` |

| Копировать строку (без выделения) | `Ctrl+C` |
| Переместить строку вверх/вниз | `Alt+↑` / `Alt+↓` |
| Дублировать строку | `Ctrl+Shift+D` |
| Удалить строку | `Ctrl+Shift+K` |
| Добавить курсор выше/ниже | `Ctrl+Alt+↑` / `Ctrl+Alt+↓` |
| Выделить все вхождения | `Ctrl+Shift+L` |
| Комментировать строку | `Ctrl+/` |

## 💡 Топ-5 для начинающих

Если запоминаешь только 5 комбинаций — запомни эти:

1. `Ctrl+L` — AI Chat (задать вопрос)
2. `Ctrl+I` — Composer (создать/изменить файлы)
3. `Ctrl+K` — быстрая правка выделенного
4. `Ctrl+P` — открыть любой файл по имени
5. `Ctrl+S` — сохранить (очевидно, но забывают)
