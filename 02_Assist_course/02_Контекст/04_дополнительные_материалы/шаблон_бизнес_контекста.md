# Шаблон: Бизнес-контекст

> Создай папку с этой структурой. AI будет обращаться к ней при каждой задаче.

## Структура

```
МОЙ_БИЗНЕС/
├── README.md              ← Краткое описание бизнеса
├── 01_О_компании/
│   ├── описание.md        ← Чем занимаемся, миссия
│   └── услуги.md          ← Что продаём, цены
├── 02_Клиенты/
│   └── портрет_ЦА.md      ← Кто клиенты, их боли
├── 03_Маркетинг/
│   └── контент_план.md    ← Каналы, тон, стратегия
├── 04_Команда/
│   └── структура.md       ← Кто за что отвечает
└── 00_Inbox/               ← Входящие
```

## README.md в корне (пример)

```markdown
# [Название компании]

**Сфера:** [что делаем]
**Клиенты:** [для кого]
**Команда:** [сколько человек]
**Выручка:** [порядок, если хочешь указать]

## Чем отличаемся от конкурентов
[2-3 пункта]

## Текущие приоритеты
1. [Приоритет 1]
2. [Приоритет 2]
3. [Приоритет 3]
```

## Промпт для автоматического создания

```
Я занимаюсь [описание бизнеса в 2-3 предложениях].
Мои клиенты — [кто].
В команде [X] человек.
Основные каналы продаж: [какие].

Создай структуру папок для бизнес-контекста.
В каждой папке — README.md с описанием.
Заполни файлы на основе того, что я рассказал.
Где не хватает информации — задай вопросы.
```
