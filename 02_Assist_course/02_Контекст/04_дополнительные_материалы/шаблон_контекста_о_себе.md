# Шаблон: Контекст о себе (Тело / Разум / Дух)

> Скопируй, заполни своими данными, сохрани как `КОНТЕКСТ_О_СЕБЕ.md` в проекте.
> Не нужно писать роман — по 3-5 пунктов в каждом блоке.
> Пиши честно: не «Война и мир», а то что реально зацепило.

```markdown
# Контекст: кто я

## 🏃 Тело

### Здоровье
- Хронические болезни / аллергии: [если есть]
- Последний чекап: [когда, основные показатели]
- Что нельзя: [ограничения по здоровью]

### Спорт
- Чем занимаюсь: [бег / зал / йога / ничего]
- Частота: [сколько раз в неделю]
- Травмы / ограничения: [если есть]

### Энергия
- Сон: [сколько часов, жаворонок или сова]
- Когда пик продуктивности: [утро / день / вечер]
- Что убивает энергию: [что выматывает]

---

## 🧠 Разум

### Психотип (после тестов)
- MBTI: [например ENTJ]
- Big Five: [ключевые черты]
- Стиль решений: [аналитик / интуит / смешанный]

### Вкусы (по 3 штуки)
- Фильмы: [1, 2, 3]
- Книги: [1 + почему, 2 + почему, 3 + почему]
- Музыка: [1, 2, 3]
- Бренды: [1, 2, 3]

### Характер
- Сильные стороны: [3 штуки]
- Что бесит: [3 триггера]
- Как общаюсь: [формально / свободно / с юмором]

---

## ✨ Дух

### Цели
- На месяц: [конкретная цель]
- На полгода: [конкретная цель]
- На год: [конкретная цель]

### Ценности
- Что важно: [3 ценности]
- Куда иду: [направление жизни]
- Чего избегаю: [что точно не хочу]

### Стратегия
- Как принимаю решения: [быстро / долго думаю / советуюсь]
- Приоритет: [скорость / качество / баланс]
- Главный принцип: [одна фраза]
```

---

## Как заполнить за 5 минут (если лень писать самому)

Открой AI Chat и напиши:

```
Задай мне 15 вопросов, чтобы понять мою личность,
стиль работы и приоритеты. Потом заполни файл
КОНТЕКСТ_О_СЕБЕ.md по структуре: Тело / Разум / Дух.
```

AI спросит — ты ответишь — он заполнит. Проще, чем писать самому.
