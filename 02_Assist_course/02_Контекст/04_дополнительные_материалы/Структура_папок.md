# Структура папок — «кабинетная система»

> Папка = кабинет. Файл = документ. README = табличка на двери.

## Для жизни

```
МОЯ_ЖИЗНЬ/
├── 00_Inbox/              ← Приёмная. Кидай сюда всё
├── 01_Здоровье/
│   ├── README.md
│   ├── анализы/
│   ├── тренировки/
│   └── питание/
├── 02_Учёба/
│   ├── README.md
│   └── курсы/
├── 03_Путешествия/
│   ├── README.md
│   └── идеи_поездок/
├── 04_Финансы/
│   └── README.md

    └── .cursorrules
```

## Для Cursor-проекта

```
МОЙ_ПРОЕКТ/
├── .cursorrules           ← Правила (AI читает всегда)
├── .cursor/
│   └── rules/
│       ├── код.mdc        ← Правила для кода
│       └── тексты.mdc     ← Правила для контента
├── README.md              ← Описание проекта
├── КОНТЕКСТ_О_СЕБЕ.md     ← Кто ты (Тело / Разум / Дух)
├── src/                   ← Исходный код
├── docs/                  ← Документация
├── data/                  ← Данные
└── output/                ← Результаты
```

## Как создать за 1 минуту

1. Скопируй нужную структуру
2. Нажми `Cmd+I` (Composer)
3. Вставь структуру + напиши: «Создай эту структуру папок. В каждой папке — README.md»
4. Accept

## Правило 00_Inbox

Не думай «куда положить» — кинь в 00_Inbox. Раз в неделю:

```
Разложи файлы из 00_Inbox по нужным папкам
```

AI сам определит куда что положить.
