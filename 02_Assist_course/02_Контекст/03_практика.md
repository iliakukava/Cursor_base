# Практика — Модуль 02. Контекст

> Задания идут от простого к сложному. Делай по порядку.

---

## Задание 1. Контекст vs без контекста

Задай один и тот же запрос двумя способами:

**Без контекста:**
```
Напиши текст для секции "О нас" на сайте
```

**С контекстом** (подключи свой .cursorrules):
```
@Codebase Напиши текст для секции "О нас" на сайте
```

Сохрани оба ответа рядом. Сравни — какой конкретнее?

✅ **Готово, когда:** Видишь, что второй ответ учитывает твой бизнес, аудиторию, тон.

## Задание 2. Структура папок для жизни

Создай отдельную папку и открой её в Cursor. В Composer (`Cmd+I`) напиши:

```
Создай структуру папок для моей жизни.
Мне важны: здоровье, учёба, финансы, хобби.
Спроси что ещё нужно.
В каждой папке — README.md с описанием что там хранить.
Добавь папку 00_Inbox для входящих.
```

Когда AI создаст структуру — положи в `00_Inbox` 3-5 любых файлов (заметки, ссылки, мысли) и попроси:

```
Разложи файлы из 00_Inbox по нужным папкам
```

✅ **Готово, когда:** У тебя есть структура с README в каждой папке, и AI умеет раскладывать файлы.

## Задание 3. Контекст Тело / Разум / Дух

Создай файл `КОНТЕКСТ_О_СЕБЕ.md`. Заполни три блока (по 3-5 пунктов в каждом):

**Тело:** здоровье, спорт, сон, энергия
**Разум:** как думаешь, что бесит, книги, стиль решений
**Дух:** цели, ценности, куда идёшь

Можешь заполнить сам, а можешь попросить AI помочь:
```
Задай мне 10 вопросов, чтобы понять мою личность,
стиль работы и приоритеты.
На основе ответов заполни файл КОНТЕКСТ_О_СЕБЕ.md
по структуре: Тело / Разум / Дух
```

✅ **Готово, когда:** Файл заполнен реальной информацией о тебе (не шаблонной).

## Задание 4. Психотесты

Зайди на **psytests.org** и пройди минимум 2 теста:

1. **Big Five** — 5 черт личности
2. **MBTI** — 16 типов личности

Сохрани результаты (скрин или текст) в папку `02_Тесты/`.

Попроси AI:
```
@02_Тесты
Проанализируй мои результаты тестов.
Как мне лучше всего организовать работу
с учётом моего психотипа?
```

✅ **Готово, когда:** AI даёт персонализированные рекомендации на основе твоих тестов.

## Задание 5. Бизнес-контекст

Создай папку для бизнес-контекста:

```
Я занимаюсь [опиши в 2-3 предложениях].
Мои клиенты — [кто они].
В команде [сколько] человек.

Создай структуру папок для бизнес-контекста.
В каждой папке — README.md.
```

Если у тебя пока нет бизнеса — сделай для учебного проекта (например, интернет-магазин или услуги репетитора).

✅ **Готово, когда:** У тебя есть бизнес-папка с README, и AI понимает твой бизнес.

## Задание 6. Тест контекста

Проверь — понимает ли AI тебя. Задай 3 вопроса:

```
1. Опиши мой бизнес в одном предложении. Кто мои клиенты?
2. Напиши пост для Telegram в моём стиле
3. Клиент просит скидку 30%. Что мне ответить?
```

Оцени каждый ответ:
- 🟢 Конкретно, под меня — отлично
- 🟡 Частично — нужно добавить контекст
- 🔴 Общие фразы — контекст пустой, вернись к заданиям 3-5

✅ **Готово, когда:** Минимум 2 из 3 ответов получили 🟢.

## Задание 7. Цепочка @-команд

Выполни 4 запроса, каждый с разным @-символом:

1. **@Codebase** — «Опиши структуру проекта и предложи улучшения»
2. **@Web** — «Найди 3 актуальных тренда в [твоя ниша] за 2026 год»
3. **@Doc** — добавь документацию сервиса, который используешь (Telegram API, AmoCRM и т.д.), и задай вопрос по ней
4. **@Git** — «Какие изменения были за последнюю неделю?» (если используешь Git)

✅ **Готово, когда:** Понимаешь, когда какой @-символ использовать и чем они отличаются.
